# dzadmin
>一款轻量级的，快速搭建后台管理平台的前端框架。使用dzadmin可以快速搭建后台界面，快速渲染表格数据，基本的系统函数及弹出动画。

## 三大功能
>表格渲染、后台搭建、系统方法
